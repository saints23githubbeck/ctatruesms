RECAPTCHA INFORMATION

reCAPTCHA is a free service that protects your site from spam and abuse, developed by Google. For this reason you will need a Google Account first. If you dont have, Simply Create a New Google Account. Once you have an Account with Google, log into the reCAPTCHA Admin page and setup your Application install URL. For more Details. Please check TRUE SMS BLOG.

FACEBOOK LOGIN

if you dont have a facebook app account, then you can create from here Facebook developer Account. After creating the app, please select Facebook Login option. Then select WWW option and enter this url True SMS on Site URL field. After that click on Basic option under the Settings in left and sidebar and fill the required details.
Finally, add this redirect url
https://truesms.codeglen/demo/login/facebook/callback under the Facebook Login > Settings on the left sidebar.

TWITTER LOGIN

if you dont have a twitter app account, then you can create from here Twitter developer Account. After creating the app, please select twitter Login option. Then select WWW option and enter this url True SMS on Site URL field. After that click on Basic option under the Settings in left and sidebar and fill the required details.
Finally, add this redirect url
https://truesms.codeglen/demo/login/twitter/callback on Callback URLs field and also put required information and collect API Key and Secret. from Keys and Tokens tab.

GOOGLE LOGIN

if you dont have a twitter app account, then you can create from here Google developer Console. and create a new project. After successfully created a new project, you need, you need to select the created proojects on the top menu click on OAuth Consent screen and select External option. After that, fill your website URl, privacy policy URL, etc. Then click Credentials menu option from the left side and select OAuth Client ID option. Finally , select Web Application and insert your application name, fill up this url
https://truesms.codeglen/demo/login/google/callback on Authorized redirects URL. Once you have clicked the create button, then you can get your Client ID and your client secret key

GITHUB LOGIN

if you dont have a Github OAuth Application, then you can create from here. Register a new OAuth Application. Then put the Application name, description and enter this url https://truesms.codeglen/demo/login/github/callback on Authorization callback URL field Once you have clicked on the Register Application button, then you can get your Client ID and your client secret key
